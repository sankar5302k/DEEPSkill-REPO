BONUS: Django ORM Migrations Command Reference

In Django, migrations are managed automatically through manage.py:

1. Create migration files for model changes:
   Command: python manage.py makemigrations

2. Apply pending migrations to database:
   Command: python manage.py migrate

3. Rollback to a specific historical migration:
   Command: python manage.py migrate <app_name> <previous_migration_number>
   Example: python manage.py migrate college_app 0002
